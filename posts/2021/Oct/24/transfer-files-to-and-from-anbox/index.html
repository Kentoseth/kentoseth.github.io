Transfer files to and from Anbox

Posted on Sun 24 October 2021 in Linux

بِسْمِ ٱللَّٰهِ ٱلرَّحْمَٰنِ ٱلرَّحِيمِ

Anbox makes it possible to bring Android apps to Linux devices. This short tutorial explains how to transfer files between your Linux host Operating System and Anbox.

Instructions

Access your file manager as the root user(or via the CLI as root). Navigate to:

/var/lib/anbox/data/media/0/

This is the folder where the data is stored. You can now transfer files to or from Anbox via this folder(this is the main folder and you will need to move data between the sub-folders, like Downloads or Pictures).

The only requirement is making sure permissions are correct. A screenshot is added below to show working permissions for a file that is transferred to Anbox from the host OS:
